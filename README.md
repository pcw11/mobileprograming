# 🌿 동숲(Dong-Soop) 스타일 README 템플릿

- 자연·식물·힐링 감성을 담은 ‘동숲 스타일’ 커스텀 테마
스마트 식물 관리 어플리케이션

## 🍃 소개 (Overview)
부드러운 녹색 톤과 식물 이미지를 기반으로, 사용자에게 편안하고 직관적인 식물 관리 경험을 제공합니다.
- 식물 별명·사진·메모 관리
- 물주기 일정 알림
- 날씨 정보 제공
- Google 검색 연동
- 직관적인 UI/UX
